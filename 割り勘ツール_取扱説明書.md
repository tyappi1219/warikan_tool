# 📱 割り勘ツール 取扱説明書

みんなで割り勘するとき、「誰がいくら払えばいい？」「端数はどうする？」と困ったことはありませんか？

このツールを使えば、**自動で公平に計算**して、**誰が誰にいくら払うか**まで教えてくれます！

---

## 🚀 はじめ方

### ツールを開く

`index.html` ファイルをダブルクリックするだけ！

ブラウザが開いて、すぐに使えます。

> 💡 **ポイント**: インターネット接続がなくても使えます

---

## 📖 基本的な使い方

### ステップ1️⃣ パーティを作成する

1. **「+ 新規作成」** ボタンをクリック
2. パーティ名（例：「新年会」「ランチ会」）を入力
3. 日付を選択

![ホーム画面イメージ]
```
┌─────────────────────────────┐
│  💰 割り勘ツール            │
├─────────────────────────────┤
│  パーティ一覧               │
│                             │
│  ┌─────────────────────┐   │
│  │ 新年会    2026-01-29│   │
│  │ 👥 5人  📝 8件      │   │
│  └─────────────────────┘   │
│                             │
│  [+ 新規作成]               │
└─────────────────────────────┘
```

---

### ステップ2️⃣ 参加者を追加する

1. 左側の「参加者」パネルで **「+」** ボタンをクリック
2. 名前を入力（例：「佐藤」「田中」）
3. 好きな色を選ぶ（誰が誰かわかりやすくなります）
4. **「追加」** ボタンをクリック

> 💡 **ヒント**: キーボードの **N キー** を押しても追加できます

```
参加者の例：
🟡 佐藤
🟢 田中  
🔵 鈴木
```

---

### ステップ3️⃣ 費用（アイテム）を追加する

1. 右側の「アイテム」パネルで **「+」** ボタンをクリック
2. 必要な情報を入力：

| 項目 | 入力例 | 説明 |
|------|--------|------|
| 品名 | ビール | 何の費用か |
| 金額 | 700 | 1つあたりの値段 |
| 数量 | 3 | いくつ頼んだか |
| 税率 | 10% | 通常は10%、食品は8% |
| 支払者 | 佐藤 | 誰が立て替えたか |
| 按分 | 等分 | みんなで割る場合 |

3. **「追加」** ボタンをクリック

> 💡 **ヒント**: キーボードの **I キー** を押しても追加できます

---

### ステップ4️⃣ 精算結果を見る

1. 画面下の **「精算結果を見る」** ボタンをクリック
2. 結果が表示されます：

```
■ 個人別内訳
┌──────────────────────────┐
│ 🟡 佐藤                   │
│ 支払額：¥5,000           │
│ 負担額：¥2,500           │
│ 差額：+¥2,500（受取）    │
└──────────────────────────┘

■ 送金指示
┌──────────────────────────┐
│ 🟢 田中 → 🟡 佐藤        │
│            ¥1,250        │
├──────────────────────────┤
│ 🔵 鈴木 → 🟡 佐藤        │
│            ¥1,250        │
└──────────────────────────┘
```

これで **「誰が」「誰に」「いくら払えばいいか」** が一目でわかります！

## 🎯 便利な機能

### 一部の人だけで割り勘したいとき

アイテム追加時に：
1. 「按分」を **「個別選択」** に変更
2. 負担する人だけにチェックを入れる

```
例：タクシー代を田中と鈴木だけで割る
    ☐ 佐藤
    ☑ 田中
    ☑ 鈴木
```

---

### 税率を変えたいとき

食品（お弁当など）は **8%**、それ以外は **10%** を選びます。

| 品目 | 税率 |
|------|------|
| お酒、外食 | 10% |
| お弁当、お菓子 | 8% |
| タクシー、備品 | 10% |
| 税込で入力したい | 0% |

---

### 端数の処理を変えたいとき

⚙️（設定）ボタンで変更できます：

| 設定 | おすすめの場面 |
|------|----------------|
| 1円単位・四捨五入 | 正確に計算したいとき |
| 10円単位・四捨五入 | 細かすぎると困るとき |
| 100円単位・切り上げ | ざっくり計算したいとき |

---

## 📤 結果を共有する

### 方法1：テキストでコピー（おすすめ）

1. **「テキストコピー」** ボタンをクリック
2. LINEやメールに貼り付けて送信

```
【新年会】
日付: 2026-01-29
合計: ¥7,500

■ 内訳
佐藤: ¥2,500 (立替: ¥5,000, 差額: +¥2,500)
田中: ¥2,500 (立替: ¥0, 差額: -¥2,500)
鈴木: ¥2,500 (立替: ¥0, 差額: -¥2,500)

■ 送金
田中 → 佐藤: ¥2,500
鈴木 → 佐藤: ¥2,500
```

### 方法2：URLで共有

1. **「URL共有」** ボタンをクリック
2. URLがコピーされます
3. 相手にURLを送ると、同じ内容が見られます

### 方法3：ファイルで保存

1. **「JSONエクスポート」** ボタンをクリック
2. ファイルがダウンロードされます
3. 次回「インポート」で読み込めます

## ❓ よくある質問（FAQ）

### Q. データは保存されますか？
**A.** はい、自動で保存されます。ブラウザを閉じても、次に開いたときにデータが残っています。

### Q. スマホでも使えますか？
**A.** はい、使えます。画面は自動で見やすく調整されます。

### Q. 他の人とデータを共有したいのですが？
**A.** 「URL共有」が便利です。相手にURLを送ると、同じ内容を見ることができます。

### Q. 間違えて入力しました。修正できますか？
**A.** はい。参加者やアイテムの横にある ✏️（編集）ボタンをクリックすると修正できます。

### Q. 参加者を削除するとどうなりますか？
**A.** その人が支払者になっているアイテムは、別の人に自動で変更されます。

### Q. ネットがなくても使えますか？
**A.** はい、インターネット接続なしで使えます。ただし「URL共有」で共有したURLを開くには、相手側でネット接続が必要です。

## ⚠️ 注意事項

### データのバックアップについて

- データはお使いのブラウザに保存されています
- ブラウザのデータを削除すると、**データも消えてしまいます**
- 大事なデータは **「JSONエクスポート」** で保存しておくことをおすすめします

### 別の端末で使いたいとき

- パソコンで作ったデータは、スマホには自動では反映されません
- **「JSONエクスポート」→「インポート」** または **「URL共有」** で移動できます

---

## 🔧 困ったときは

### 画面が真っ白になった

→ ブラウザの更新ボタン（🔄）を押してください

### データが消えてしまった

→ 以前エクスポートしたJSONファイルがあれば、「インポート」で復元できます

### ボタンが押せない

→ 参加者を先に追加してください（アイテム追加は参加者が必要です）

### 計算結果がおかしい気がする

→ 以下を確認してください：
1. 税率は正しいですか？（0%/8%/10%）
2. 支払者は正しく選択されていますか？
3. 「個別選択」の場合、対象者は正しくチェックされていますか？

## 📝 計算例

### 例1：シンプルな3人割り

**状況**: 飲み会の合計 ¥9,000 を佐藤が立て替えた

| 入力 |  |
|------|---|
| 金額 | 9000 |
| 税率 | 0%（税込で入力） |
| 支払者 | 佐藤 |
| 按分 | 等分 |

**結果**:
- 佐藤：¥3,000 負担（¥6,000 受け取り）
- 田中：¥3,000 負担（佐藤に ¥3,000 支払い）
- 鈴木：¥3,000 負担（佐藤に ¥3,000 支払い）

---

### 例2：複数アイテムと立替者が異なる場合

**状況**: 
- 居酒屋代 ¥8,000（佐藤が支払い）
- 二次会代 ¥6,000（田中が支払い）
- 参加者：佐藤、田中、鈴木

**入力**:
| 品名 | 金額 | 支払者 |
|------|------|--------|
| 居酒屋 | 8000 | 佐藤 |
| 二次会 | 6000 | 田中 |

**結果**（税率0%の場合）:
- 合計：¥14,000
- 一人あたり：約 ¥4,667
- 佐藤：¥3,333 受け取り
- 田中：¥1,333 受け取り
- 鈴木：¥4,667 支払い

**送金**:
- 鈴木 → 佐藤：¥3,333
- 鈴木 → 田中：¥1,334

---

### 例3：一部の人だけ参加したアイテムがある場合

**状況**: 
- 食事代 ¥6,000（3人で割る）
- タクシー代 ¥2,000（田中と鈴木の2人だけ）

**入力**:
| 品名 | 金額 | 支払者 | 按分 | 対象者 |
|------|------|--------|------|--------|
| 食事 | 6000 | 佐藤 | 等分 | 全員 |
| タクシー | 2000 | 田中 | 個別選択 | 田中、鈴木 |

**結果**:
- 佐藤：¥2,000（食事のみ）
- 田中：¥3,000（食事 + タクシー半分）
- 鈴木：¥3,000（食事 + タクシー半分）
